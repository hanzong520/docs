# Spring-cloud

 ## Nacos启动

``` shell
sh ./startup.sh -m standalone
```

启动之后的访问地址：

http://localhost:8848/nacos/index.html# 

默认账号密码都是：nacos

## Sentinel

下载地址：https://github.com/alibaba/Sentinel/tags

下载对应版本的jar包

启动命令

```
 java -jar ./sentinel-dashboard-1.8.0.jar
```

启动之后的访问地址：

http://localhost:8080/
